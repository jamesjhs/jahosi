I wanted to let the AI run wild, so asked it simply to create a visually catching and imaginative website exemplifying the best of modern ways to display content with as many technologies as possible. Frankly, it's not my work at all, but I enjoy the "wild ride through colour, motion and interactivity — built entirely with CSS animations and vanilla JavaScript, optimised for touch devices". All the numbers are purely fabrications, but I liked its optimism!

🎨 About This Project

Neon Dreamscape explores the intersection of motion, light, and digital art. Every element is carefully choreographed to feel alive — from the drifting aurora blobs behind the page to the particles that float upward like embers.

Designed mobile-first, with touch ripples, a pinch-to-close lightbox, and a tilt-response parallax on supported devices.

⚡ Tech Stack

  • Pure HTML5 + CSS3 animations
  • Vanilla JS — zero dependencies
  • Intersection Observer for scroll-reveal
  • Touch-event ripple effect
  • DeviceOrientation tilt parallax
  • Glassmorphism + backdrop-filter
