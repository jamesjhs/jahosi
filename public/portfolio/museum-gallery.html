Aviation Museum Archives and Gallery

A Node.js-powered image gallery hosting the archive photography collection for a local Aviation Museum. This was the biggest project yet of my journey into exploring the world AI-assisted coding, and before I'd particularly heard of Agentic AI - building on my coding skills and previous knowledge, I wanted to see how easy it was to parse out an exported database (from the museum software Modes, which used a custom Nexus-derived database), to make the content browseable in 21st-Century standards as opposed to the Windows 9x-esque interface of Modes.

After a lot of iterative steps, and purely using Google Gemini for help, the website (for internal use only, so far) has actually turned out to be very much a success!
