PingMe Help

Self-hosted help & support portal · Node.js / TypeScript · Tailwind CSS

PingMe Help is a lightweight, self-hosted helpdesk and support request portal. Users can submit help tickets, track their status through to resolution, and receive email updates — all within a clean, private environment on your own server.

Designed for small teams and private deployments where a full-fat commercial helpdesk is overkill, PingMe Help keeps things simple: raise a request, assign a category and priority, watch it move from Open through to Resolved. Administrators get a full queue view with filtering and bulk actions, and every state change triggers an optional email notification to the submitter.

Key features

  • Submit help requests with category, priority, and a free-text description
  • Priority levels: Normal, High, Urgent — each colour-coded in the queue
  • Status workflow: Open → In Progress → Resolved (→ Closed)
  • Email notifications to the submitter on each status change
  • Admin queue view with full filtering and search
  • Internal notes — add private comments visible only to admins
  • Category management — configure categories and sub-categories in settings
  • Audit trail — every status change timestamped and logged
  • Multi-user: user accounts with role-based access (user / admin)
  • Clean, minimal responsive UI
