Project

QGlance

Quick-glance personal dashboard · Node.js · Tailwind CSS

QGlance is a minimal personal dashboard designed to give you a quick, at-a-glance view of the things that matter each day. No login walls, no subscription prompts — just a clean page of configurable widgets that pulls from the sources you care about and presents them without clutter.

The brief was a single browser tab you can open and immediately have a clear picture of the day ahead: calendar events, service statuses, key metrics, and recent activity from your own apps. Widgets are arranged in a configurable grid and each one is independently refreshed on a short polling interval so the page stays current without a full reload.

Key features

  • Configurable widget grid — arrange and resize tiles to suit your workflow
  • Service health widget — polls your own apps and shows live up/down status
  • Calendar widget — upcoming events pulled from an ICS feed
  • Integration widgets for other JHS projects: Leccy, Hamster Logger, Taskit
  • Auto-refresh polling — each widget updates independently on a configurable interval
  • Dark-first UI with optional light mode
  • No external dependencies at runtime — self-contained, self-hosted
  • Single-page application — everything on one tab, no navigation required
  • Settings panel to add/remove widgets and configure data sources
